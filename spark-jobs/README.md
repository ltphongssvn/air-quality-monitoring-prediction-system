# Spark Jobs

Spark Streaming, MLlib, and GraphX jobs for air quality processing.

## Prerequisites
- Java 17+
- SBT 1.9.9
- Spark 3.5.0

## Build
```bash
sbt compile
# [success] Total time: 20 s
```

## Dependencies
- Spark Core, SQL, Streaming, MLlib, GraphX
- Kafka integration
- Breeze (numerical computing)
